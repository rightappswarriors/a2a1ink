##
````
Main Application Pages
````

This directory contains the primary page views for the application, organized by functional area:

- **pages/** - Full-page views for major features (dashboard, learners, attendance, devices, categories, consumers, items, logs, settings)
- **pages/auth/** - Authentication pages (login, register)
- **pages/categories/** - Category management pages (list, create, update)
- **pages/consumers/** - Consumer management pages (list, create, update)
- **pages/devices/** - Device management pages (list, create, update, new)

These views represent complete pages that are typically rendered through controllers and wrapped with layouts to form the final user interface.
